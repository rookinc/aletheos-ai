## Thesis

A registered trace is boundary-preserving finite aftermath, not measurement, display, transport, channel, or force.

## Core rule

An object is a registered trace when all required criterion atoms hold:

1. admitted body source
2. admissible action or relation
3. finite residue or contrast
4. non-display origin
5. reproducible receipt
6. typed as trace before transport
7. boundary preserving
8. classification deferred

## Trace classes

- residue trace
- return trace
- obstruction trace
- correlation trace
- transition trace

## Important separation

This criterion outputs `registered_trace` only.

It does not output:

- transport relation
- channel
- viewer display
- physical force
- physics claim
