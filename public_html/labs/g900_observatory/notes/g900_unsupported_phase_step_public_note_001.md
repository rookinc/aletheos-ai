# What the G900 6/9 anomaly means

Status: unsupported phase step public note recorded.

Audit pass: `true`.

This is a public-facing explanatory note. It does not admit channels.

## Short version

The 6/9 anomaly is real in the G900 phase grammar, but unsupported by the tested carrier rails. It is not a channel. It marks the exact place where the current Layer 2 carrier language stops explaining the observed phase motion.

## What it means

The G900 Observatory now separates three things that are easy to confuse.

First, there is phase grammar. A phase transition is real when it appears reproducibly in the edge-limit scan. On that test, 6/9 is real.

Second, there is carrier support. A transition is supported when its step is witnessed by the tested internal or boundary carrier rails. On that test, nine of the ten full-60 packages are supported.

Third, there is channel admission. A channel would require explicit registry admission. No channel has been admitted.

The 6/9 package is the exception. It appears in the phase grammar with full 60-root support, but it is absent from the Layer 2 boundary-carrier registry and has no tested internal delta-edge support. It is therefore best read as an unsupported internal phase step inside the slot-12 composite triangle.

In plain terms: the motion is visible, but the current carrier language does not explain it.

That is not a failure. It is a useful boundary. The Observatory can now say exactly where the current carrier grammar works, and exactly where it stops.
