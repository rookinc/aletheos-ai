# Upstairs / downstairs inscription interpretation 001

Status: interpretation recorded.

## Core statement

I is the inscription station.

It is the place where the just-lived transport is written into history.

## Layer grammar

Before I: upstairs motion.

The event is still happening. The path is becoming.

At I: writing.

The event is inscribed as registered history.

After I: downstairs return.

Motion is no longer raw traversal. It is return carrying trace.

## Projection modes

Mode 1:

    W-X-Y-Z-T-I-W

Mode 1 is direct cyclic return after inscription. It is like an elevator return from I to W.

Mode 2:

    W-X-Y-Z-T-I-T-Z-Y-X-W

Mode 2 is sheet-flip return. It is like taking the stairs back down through the house. The return passes back through the seams it crossed, but now the motion is history-bearing.

## Role grammar

- W: witness / doorway
- X: first hinge
- Y: upper passage
- Z: lower passage
- T: threshold
- I: inscription station / stairwell between becoming and history

## Boundary

This interpretation does not change the G60 source object.

It does not claim physical time reversal.

It does not replace the validation receipts.

It is role grammar for the validated projection modes.
