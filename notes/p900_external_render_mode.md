# P900 External Render Mode

Status: browser renderer bridge

Adds a separate `P900 External` render mode to the existing constructor lab.

Boundary:

    This mode loads public P900 JSON artifacts.
    It does not change the D4 growth engine.
    It renders the external P900 scaffold only.
    It does not include internal G60 thalion edges.

Primary data:

    /json/p900/p900_phase17_external_edge_list.json
    /json/p900/p900_phase20_checkpoint_summary.json
